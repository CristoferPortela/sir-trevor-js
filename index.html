Good evening. I'm Sir Trevor.

I'm a rich content editor re-imagined for the web

I have a beautiful, intuitive interface for creating content

And finally... I'm easy to extend and customise

Sir Trevor is rich content editing entirely re-imagined for the web: an intuitive editor for web content which does not presuppose anything about how it will be rendered.

Get started See an example

Features

  • No HTML is stored, only structured JSON and clean Markdown
  • Trivial to extend and create your own block types
  • Intuitive interface for creating rich content
  • Used on a national broadcast news site serving millions of users

Block types

Sir trevor content is made up of blocks. By default Sir Trevor ships with the following block types, but you can easily add your own

  • text Text
  • image Image
  • quote Quote
  • list List
  • twitter Tweet
  • video Video

There are more blocks available in the dedicated blocks repository or it's incredibly easy to write your own.

For more information on Sir Trevor JS check out the repository at github.com/madebymany/sir-trevor-js
